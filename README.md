## Used technologies:

- Pug
- Sass
- Vanilla js
- Gulp

## Run project:

1. clone repository
   `git clone https://github.com/x4AEKx/Chairs.git your-folder`

1. make sure GULP is installed on your computer
   for linux install:
   `sudo apt install gulp`

1. install dependencies
   `npm install`

1. for run project type:
   `gulp serve`
